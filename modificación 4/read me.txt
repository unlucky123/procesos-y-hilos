Modifica la clase CFilPrincipal, de forma que ésta inicie la ejecución de
tantos procesos hijos como indique el usuario (el usuario puede indicar este
número mediante un parámetro del procedimiento “main” o mediante consola).
Además:
● Utiliza el método correspondiente de la clase cFil para que cada proceso
hijo iniciado por el proceso padre tenga una temporización diferente (el
primer proceso hijo, la más pequeña, y así sucesivamente hasta el último
proceso hijo, que tendrá la mayor temporización).
● Haz que el proceso padre no acabe hasta que todos sus procesos hijos no
hayan terminado.
● Implementa un límite superior para que el usuario no pueda introducir un
número de procesos hijos exageradamente grande.
